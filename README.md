##🔐 Password Generator :

- Generates strong passwords with:
  - ✅ Uppercase letters
  - ✅ Lowercase letters
  - ✅ Numbers
  - ✅ Symbols
- Ensures minimum security by including at least one character from each category
- Customizable password length
- Console-based (easy to run on any system)
